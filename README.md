End-to-end project using CMS Medicare public use data to build:

\- Databricks Bronze/Silver/Gold tables (Delta)

\- dbt transformations + tests + documentation

\- AWS SageMaker regression + time series models

\- Power BI dashboards using curated tables + predictions
